JavaScript - Lenguaje de Programación

El lenguaje que añade interactividad y comportamiento dinámico a las páginas web

¿Qué es JavaScript?

JavaScript es el lenguaje de programación de la web. Permite crear contenido dinámico, controlar multimedia, animar imágenes, validar formularios, y prácticamente todo lo que involucre interacción con el usuario. Es el tercer pilar fundamental del desarrollo web junto con HTML y CSS.

Creado en 1995, JavaScript ha evolucionado enormemente. Hoy es uno de los lenguajes más populares del mundo, usado tanto en el navegador (frontend) como en servidores (backend con Node.js), aplicaciones móviles, desktop, IoT y más.

Características principales

Manipulación del DOM

Modifica el contenido, estructura y estilos de la página en tiempo real.

Manejo de Eventos

Responde a acciones del usuario como clicks, formularios, teclas y más.

Asincronía

Realiza múltiples tareas sin bloquear la interfaz usando Promises y async/await.

APIs del Navegador

Accede a funcionalidades como geolocalización, almacenamiento local y más.

Manipulación del DOM

El DOM (Document Object Model) es la representación en JavaScript del HTML. Puedes seleccionar y modificar elementos:

Seleccionar y modificar elementos JavaScript 
// Seleccionar elementos
const titulo = document.querySelector('h1');
const botones = document.querySelectorAll('.btn');
const form = document.getElementById('contact-form');

// Modificar contenido
titulo.textContent = "Nuevo título";
titulo.innerHTML = "<strong>Título en negrita</strong>";

// Modificar estilos
titulo.style.color = "#0ea5ff";
titulo.style.fontSize = "2rem";

// Agregar/quitar clases
titulo.classList.add('activo');
titulo.classList.remove('inactivo');
titulo.classList.toggle('destacado');

// Crear elementos nuevos
const nuevoParrafo = document.createElement('p');
nuevoParrafo.textContent = "Nuevo párrafo";
document.body.appendChild(nuevoParrafo);

// Eliminar elementos
const elemento = document.querySelector('.eliminar');
elemento.remove();

Manejo de Eventos

Los eventos permiten que tu código responda a acciones del usuario:

Event Listeners JavaScript 
// Click en un botón
const boton = document.querySelector('.btn');
boton.addEventListener('click', () => {
    alert('¡Botón clickeado!');
});

// Submit de formulario
const formulario = document.querySelector('#contact-form');
formulario.addEventListener('submit', (evento) => {
    evento.preventDefault(); // Prevenir envío por defecto

    const nombre = document.querySelector('#nombre').value;
    const email = document.querySelector('#email').value;

    console.log(`Nombre: ${nombre}, Email: ${email}`);
});

// Hover sobre elemento
const tarjeta = document.querySelector('.card');
tarjeta.addEventListener('mouseenter', () => {
    tarjeta.style.transform = 'scale(1.05)';
});

tarjeta.addEventListener('mouseleave', () => {
    tarjeta.style.transform = 'scale(1)';
});

// Tecla presionada
document.addEventListener('keydown', (e) => {
    if (e.key === 'Escape') {
        cerrarModal();
    }
});

// Scroll de la página
window.addEventListener('scroll', () => {
    const scrollY = window.scrollY;
    console.log(`Scroll: ${scrollY}px`);
});
click

Se dispara cuando el usuario hace click en un elemento

submit

Se dispara cuando se envía un formulario

change / input

Se dispara cuando cambia el valor de un input

keydown / keyup

Se dispara cuando se presiona o suelta una tecla

mouseenter / mouseleave

Se dispara cuando el mouse entra o sale de un elemento

Validación de formularios

Ejemplo práctico de validación de un formulario de contacto:

Validar formulario JavaScript 
const form = document.querySelector('#contact-form');
const nombreInput = document.querySelector('#nombre');
const emailInput = document.querySelector('#email');
const mensajeInput = document.querySelector('#mensaje');
const feedback = document.querySelector('#form-feedback');

form.addEventListener('submit', (e) => {
    e.preventDefault();

    // Limpiar errores previos
    limpiarErrores();

    let errores = [];

    // Validar nombre
    if (nombreInput.value.trim().length < 2) {
        errores.push('El nombre debe tener al menos 2 caracteres');
        nombreInput.classList.add('is-error');
    }

    // Validar email
    const emailRegex = /^[^\s@]+@[^\s@]+\.[^\s@]+$/;
    if (!emailRegex.test(emailInput.value)) {
        errores.push('El email no es válido');
        emailInput.classList.add('is-error');
    }

    // Validar mensaje
    if (mensajeInput.value.trim().length < 10) {
        errores.push('El mensaje debe tener al menos 10 caracteres');
        mensajeInput.classList.add('is-error');
    }

    // Mostrar errores o éxito
    if (errores.length > 0) {
        mostrarErrores(errores);
    } else {
        mostrarExito();
        enviarFormulario();
    }
});

function limpiarErrores() {
    nombreInput.classList.remove('is-error', 'is-success');
    emailInput.classList.remove('is-error', 'is-success');
    mensajeInput.classList.remove('is-error', 'is-success');
    feedback.hidden = true;
}

function mostrarErrores(errores) {
    feedback.hidden = false;
    feedback.className = 'form-feedback feedback-error';
    feedback.innerHTML = errores.join('<br>');
}

function mostrarExito() {
    feedback.hidden = false;
    feedback.className = 'form-feedback feedback-success';
    feedback.textContent = '¡Formulario enviado con éxito!';

    nombreInput.classList.add('is-success');
    emailInput.classList.add('is-success');
    mensajeInput.classList.add('is-success');
}

function enviarFormulario() {
    // Aquí iría el código para enviar al servidor
    console.log('Enviando formulario...');
}

Fetch API - Peticiones HTTP

Realiza peticiones a servidores para obtener o enviar datos:

Fetch con async/await JavaScript 
// GET: Obtener datos
async function obtenerUsuarios() {
    try {
        const respuesta = await fetch('https://api.ejemplo.com/usuarios');

        if (!respuesta.ok) {
            throw new Error('Error en la petición');
        }

        const datos = await respuesta.json();
        console.log(datos);
        mostrarUsuarios(datos);

    } catch (error) {
        console.error('Error:', error);
        mostrarError('No se pudieron cargar los usuarios');
    }
}

// POST: Enviar datos
async function crearUsuario(usuario) {
    try {
        const respuesta = await fetch('https://api.ejemplo.com/usuarios', {
            method: 'POST',
            headers: {
                'Content-Type': 'application/json',
            },
            body: JSON.stringify(usuario)
        });

        const datos = await respuesta.json();
        console.log('Usuario creado:', datos);

    } catch (error) {
        console.error('Error:', error);
    }
}

// Uso
obtenerUsuarios();

crearUsuario({
    nombre: 'Ana García',
    email: 'ana@ejemplo.com'
});

¿Por qué es importante JavaScript?

1

Interactividad

JavaScript convierte sitios estáticos en aplicaciones web interactivas y dinámicas que responden a las acciones del usuario.

2

Experiencia de usuario

3

Versatilidad

Un solo lenguaje para frontend, backend, móvil, desktop y más. Es la base del desarrollo web moderno.
