HTML - HyperText Markup Language

El lenguaje fundamental de la web que define la estructura y el contenido de las páginas

¿Qué es HTML?

HTML (HyperText Markup Language) es el lenguaje de marcado estándar utilizado para crear y estructurar contenido en la web. No es un lenguaje de programación, sino un lenguaje de marcado que utiliza etiquetas para definir diferentes elementos de una página web.

Fue creado por Tim Berners-Lee en 1991 y ha evolucionado hasta HTML5, la versión actual que incluye soporte para multimedia, gráficos y aplicaciones web modernas.

Características principales

Estructura Semántica

Usa etiquetas que describen el significado del contenido, no solo su apariencia.

Accesibilidad

Permite crear sitios accesibles para todos los usuarios, incluyendo personas con discapacidades.

SEO Optimizado

Los motores de búsqueda comprenden mejor el contenido estructurado correctamente.

Estándar Universal

Funciona en todos los navegadores y dispositivos sin necesidad de plugins.

Etiquetas semánticas importantes

HTML5 introdujo etiquetas semánticas que describen claramente el propósito de cada sección:

Estructura básica semántica HTML 
<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <title>Mi Sitio Web</title>
</head>
<body>
    <header>
        <!-- Encabezado de la página -->
        <nav>
            <!-- Menú de navegación -->
        </nav>
    </header>

    <main>
        <!-- Contenido principal -->
        <article>
            <!-- Contenido independiente -->
            <section>
                <!-- Sección temática -->
            </section>
        </article>

        <aside>
            <!-- Contenido relacionado -->
        </aside>
    </main>

    <footer>
        <!-- Pie de página -->
    </footer>
</body>
</html>
<header>

Encabezado de la página o de una sección

<nav>

Bloque de navegación principal del sitio

<main>

Contenido principal único de la página

<article>

Contenido independiente que puede ser reutilizado

<section>

Agrupación temática de contenido

<aside>

Contenido tangencialmente relacionado

<footer>

Pie de página o de una sección

¿Por qué es importante HTML?

1

Base sólida

Un HTML bien estructurado es la base de cualquier sitio web profesional y facilita el mantenimiento del código.

2

Mejor accesibilidad

Las etiquetas semánticas ayudan a los lectores de pantalla y otras tecnologías asistivas a interpretar el contenido.

3

SEO mejorado

Los motores de búsqueda priorizan sitios con HTML semántico y bien estructurado.
