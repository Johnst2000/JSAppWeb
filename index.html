¿Qué es el diseño web?

Es la disciplina que combina estructura, estética y usabilidad para crear experiencias digitales accesibles y agradables. Se fundamenta en tres tecnologías principales.

HTML

Estructura el contenido usando etiquetas semánticas.

CSS

Define estilos, diseño responsivo y apariencia visual.

JavaScript

Permite interactividad, validación y dinámicas de página.

Principios de diseño web

  • Jerarquía: guía visual clara.
  • Contraste: mejorar lectura y accesibilidad.
  • Consistencia: patrones homogéneos.
  • Accesibilidad: navegación por teclado, ARIA, textos alternativos.
  • Inclusividad: diseño para personas con diferentes capacidades.
  • Rendimiento: optimización de recursos.
